Prework Study Guide

HTML

  • Provides content and structure to a webpage - the foundation of the house
  • The head element contains information for the browser not visable to the user
  • The body element contains what the user will see and interact with
  • Angled brackets contain elements
  • Most elements are made up of an opening and closing tag
  • Nested elements are elements contained in other elements
  • Nested elements are referred to as child elements and the elements they are nested in the parent element
  • Elements have attributes - this is additional information about the element
  • Attributes are always in the opening tag never the closing tag
  • Semantic HTML - The name of the element describes what it does

CSS

  • Provides style for the webpage - the decor of the house

Git

  • Accessed through the CLI
  • git status: shows what branch we are currently working on
  • git checkout -b branch name: creates a new branch and switched to it

Javascript

  • Provides functionaliy for the webpage - you can now open doors in the house!
